# Personal Expense Tracker

A simple Python project to track your daily expenses with categories and monthly summaries.

## Features
- Add expenses with amount, category, description, and date
- View all expenses
- Filter expenses by category
- Monthly summary with total spending
- Data stored locally in JSON file

## How to Run
1. Clone the repo
2. Run `python expense_tracker.py`
3. Follow the menu to manage your expenses
